cy3sabiork

cy3sabiork provides access to SABIO-RK information in SBML format.
SABIO-RK webservice query Get SABIO-RK results from the webservice.

More information about the SABIO-RK webservices is available from http://sabiork.h-its.org/layouts/content/webservices.gsp.
