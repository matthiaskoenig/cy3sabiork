cy3sabiork

cy3sabiork provides access to SABIO-RK information in SBML format via web services.
SABIO-RK webservice query Click the SABIO-RK icon to perform a query.

Information about the SABIO-RK webservices is available at http://sabiork.h-its.org/layouts/content/webservices.gsp.
